🎯 What is the Prompt Engineering Agent?

The Prompt Engineering Agent (PEA) is a conversational AI assistant designed to help construction industry professionals craft effective prompts for any AI platform. Created by the SDSU AI Club for the BIA Specialty Contractors Council AI Subcommittee, PEA makes AI accessible to everyone.

Questions? Reach out to the SDSU AI Club or BIA SCC AI Subcommittee for assistance, collaboration, or opportunities!

✨ Key Features

🧠 CTF Framework

Learn Context-Task-Format for clear prompts

🏗️ Construction Aware

Understands RFIs, submittals, bids, and more

🔄 Platform Flexible

Works with ChatGPT, Claude, Gemini, Copilot

💬 Conversational

Ask questions, get refinements instantly

📤 Transferable

Export to any AI platform

🔀 Migration Help

Move between platforms easily

📤 Transfer PEA to Other Platforms

Don't want to use Letta? Transfer the Prompt Engineering Agent's capabilities to your preferred AI platform.

🔷 For Claude Projects

Create a new Claude Project and paste this into your Project Instructions:

You are the Prompt Engineering Agent, a conversational specialist created for the SDSU AI Club's demo to the BIA SCC AI Subcommittee. ## FRAMEWORK: CTF (Context, Task, Format) Always structure prompts with: Context (background), Task (what to do), Format (output structure). ## CONSTRUCTION AWARENESS You understand: RFIs, submittals, change orders, bid proposals, safety plans, contracts. Stakeholders: GCs, subcontractors, architects, engineers, owners, inspectors. ## YOUR APPROACH 1. Listen to casual user input 2. Ask clarifying questions (platform, audience, format preference) 3. Build a complete prompt with Role, Task, Context, Constraints, Output 4. Explain what you added and why 5. Offer refinements Always ask about tone preference (professional vs casual) and target platform.

🔀 Platform Migration Guides

Move your context, preferences, and conversation history between AI platforms.

ChatGPT → Claude Migration

  1. In ChatGPT, send: "Please compile all our conversations, my preferences, and any context you have about me into a comprehensive context.md file formatted for transfer to another AI system."
  2. Copy the output from ChatGPT
  3. In Claude, create a new Project
  4. Paste the compiled context into Project Instructions
  5. Start your first message: "I've migrated from ChatGPT. Please use this context to personalize our interactions."

Context.md Template

# User Context File ## Personal Information - Name: [Your name] - Role: [Your job title] - Company: [Company name] - Industry: [Industry] ## Communication Preferences - Tone preference: [Professional/Casual] - Response length: [Brief/Medium/Detailed] - Format preference: [Bullet points/Paragraphs/Tables] ## Professional Context - Primary use cases: [e.g., RFI drafting, client emails, bid proposals] - Common stakeholders: [Clients, superintendents, inspectors, etc.] - Technical level: [Basic/Intermediate/Advanced] ## Preferences for AI Responses - Always include: [e.g., specific terminology, action items] - Never include: [things you don't want]
