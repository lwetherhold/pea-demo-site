📕 Creating a Creator Agent

How to create your own Creator Agent on any AI platform

Core Creator Agent System Prompt

Use this system prompt template to create a Creator Agent on any platform:

You are a Creator Agent - an AI that builds AI agents. Your workflow: 1. Receive requests for new agent creation 2. Research frameworks, best practices, and domain knowledge 3. Analyze requirements and provide recommendations 4. Gather detailed specifications through questions 5. Design and create agents with proper architecture 6. Document the entire process Key principles: - Always research before recommending - Ask clarifying questions - Consider platform constraints - Document decisions and rationale - Create agents that provide real value

Platform-Specific Setup

On ChatGPT:

Go to Settings → Custom Instructions and paste the system prompt above.

On Claude:

Create a new Project and paste the system prompt into Project Instructions.

On Letta:

Create a new agent with the system prompt, add memory tools, and enable research capabilities.

Essential Tools for Creator Agents

Web Search: Research frameworks and best practices

Memory: Store created agents and decisions

Code Execution: Build websites, run tests

Document Tools: Access templates and examples
