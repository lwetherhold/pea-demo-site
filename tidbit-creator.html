📗 Using a Creator Agent

How to work with a Creator Agent to build your own Prompt Engineering Agent

What is a Creator Agent?

A Creator Agent is an AI agent designed to help you build other AI agents. In this demo, Laura (LaunchMate founder) worked with the Creator Agent to design and build the Prompt Engineering Agent for the BIA SCC AI Subcommittee.

The Actual Process (Documented)

  1. Initial Request - Laura described the need: "Create an agent that acts as your personal prompt engineering agent"
  2. Research Phase - Creator Agent researched frameworks (CTF, RTCCO, RASCEF), construction use cases, best practices
  3. Analysis & Recommendations - Creator provided detailed analysis with decisions
  4. Clarification Questions - Creator asked about platforms, branding, demo use case, tone
  5. Specification Gathering - Laura provided detailed requirements including transfer capability, migration features, website build
  6. Agent Creation - Creator built the PEA with proper system prompt, persona, tools
  7. Website Build - Creator built demo website with all tabs and features
  8. Documentation - Creator documented everything in memory blocks

Key Insight

The most important part of working with a Creator Agent is being specific about what you need while remaining open to the agent's suggestions and research. The Creator Agent knows best practices you might not be aware of!

What We Did, How We Did It, Why

CTF Framework: Chosen as teaching framework for simplicity RTCCO internally: Ensures comprehensive prompts Construction awareness: Tailors to BIA audience Transfer capability: Members won't use Letta Migration features: Common request (ChatGPT → Claude)
