📘 How to Use Letta

Getting started from scratch with Letta and the Prompt Engineering Agent

What is Letta?

Letta is an AI agent platform that lets you create and interact with AI agents that have persistent memory. Unlike regular chatbots, Letta agents remember your conversations, preferences, and context across sessions.

Step-by-Step Setup

  1. Visit Letta - Go to letta.com in your web browser
  2. Create Account - Click "Sign Up" and create your free account with email or Google
  3. Verify Email - Check your email and click the verification link
  4. Log In - Return to Letta and log in with your credentials
  5. Explore Dashboard - Familiarize yourself with the Letta interface
  6. Access PEA - Use the PEA agent file or template provided by SDSU AI Club
  7. Start Chatting - Begin your conversation with the Prompt Engineering Agent!

Key Terms to Know

Agent: An AI assistant with memory capabilities

Memory Block: Persistent storage for information about you or the agent

Archival Memory: Long-term storage for documents and knowledge

Tools: Capabilities the agent can use (web search, memory, etc.)
